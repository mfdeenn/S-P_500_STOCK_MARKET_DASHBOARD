S&P 500 Stock Market Analysis Dashboard

This project is an interactive financial analytics dashboard built in Power BI to analyze historical performance of 505 S&P 500 companies using over 1 million stock market records.

It provides insights into stock performance, market volatility, trading volume trends, and risk vs return relationships.

Project Objectives

The goal of this project is to:

Analyze long-term stock performance trends

Compare major tech giants over time

Identify top-performing stocks

Understand market volatility patterns

Evaluate risk vs return trade-offs

Track market trading volume over time
